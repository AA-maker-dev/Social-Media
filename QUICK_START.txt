**Result:** Your code is on GitHub ✅

---

## STEP 2️⃣ : Deploy on Vercel (5 minutes total)

### Frontend Deployment
1. Go to https://vercel.com → Sign in with GitHub
2. Click "New Project"
3. Select "Social-Media" 
4. **Settings:**
   - Root: `./Social-Media/Website_Inside/HTML`
   - Framework: Other
   - Click "Deploy"

**Result:** Frontend deployed ✅

### Backend Deployment  
1. Click "New Project" again
2. Select "Social-Media" again
3. **Settings:**
   - Root: `./backend`
   - Build: `npm install`
   - Framework: Other
4. **Add Variables:**
   - `NODE_ENV=production`
   - `CORS_ORIGIN=https://viteflo.com`
   - `PORT=3001`
5. Click "Deploy"

**Result:** Backend deployed ✅

---

## STEP 3️⃣ : Connect Domain (5 minutes setup + 24-48 hours waiting)

### In Vercel Dashboard:
1. Frontend Project → Settings → Domains → Add: `viteflo.com`
2. Backend Project → Settings → Domains → Add: `api.viteflo.com`
3. Copy the **nameservers** Vercel shows

### At nepalownregistrar.com:
1. Log in to your account
2. Find domain settings for viteflo.com
3. Go to DNS/Nameserver settings
4. Replace with Vercel nameservers:
   ```
   ns1.vercel-dns.com
   ns2.vercel-dns.com
   ns3.vercel-dns.com
   ns4.vercel-dns.com
   ```
5. Save

**Result:** Domain connected ✅  
**Wait:** 24-48 hours for DNS to propagate ⏳
