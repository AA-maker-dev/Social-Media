✅ POST CREATION BUG - FIXED

═══════════════════════════════════════════════════════════════════════════

🐛 ISSUE:
   Cannot add posts on the home page - post button not responding to clicks

🔍 ROOT CAUSE:
   Post button click event listener was inside an IndexedDB promise's .catch()
   block, meaning it only attached if there was an error loading IndexedDB.
   In normal operation (no error), the listener was never attached.

✅ SOLUTION:
   Moved the post button event listener outside the promise chain so it
   always gets attached regardless of IndexedDB state.
